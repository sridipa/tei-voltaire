# TEI Voltaire — Candide Encoding Project

A digital scholarly edition of selections from Voltaire's *Candide* (1759),
built as a practice project in TEI P5 encoding, XSLT transformation, and 
RDF metadata modelling.

## Structure

- `tei/` — TEI P5 XML encoding with full teiHeader and named entity annotation
- `xslt/` — XSLT 2.0 stylesheet transforming TEI → HTML
- `output/` — HTML output rendered from the transformation
- `rdf/` — Dublin Core RDF/XML metadata extracted via XSLT

## Tools

oXygen XML Editor · Git · VS Code
